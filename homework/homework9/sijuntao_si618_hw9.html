Question 2: Day-of-the-week Effect (15 points)¶

Is there a day-of-the-week effect on Wikipedia? On average, which day sees the most Wikipedia views (total across all articles)? What about the least views?

In [5]:
per_day_views = views.sum()
weekday_views = pd.DataFrame({ "weekday": [pd.to_datetime(date).strftime('%A') for date in per_day_views.index], "tot_view": list(per_day_views)})
q2 = weekday_views.groupby("weekday").mean().round().astype(int).sort_values('tot_view', ascending=False)
q2 = q2.reset_index().rename(columns = {"weekday":"day", "tot_view":"views"})
print(q2)

From the table above, we can find that:

  • There are more views in the previous half of a week (Sunday, Monday, Tuesday and Wednesday) while less views in the latter half of a week (Thursday, Friday and Saturday). Therefore, there is a day-of-the-week effect on Wikipedia.
  • Monday sees the most Wikipedia views on average
  • Friday sees the least Wikipedia views on average

Question 3: Wikipedia views per day (30 points)¶

Plot Wikipedia views per day (total across all pages) over time. Then plot the de-seasonified data where the day-of-the-week effects are removed. Put those in the same plot using grid.arrange. (Note: You will need to change the figure width to see the plots properly. You dont have to match the exact same width we have here, just make sure it is readable.)

Hint: Use mizani package to get the correct formatting for the axis labels

day_views = pd.DataFrame({"date": [pd.to_datetime(date) for date in per_day_views.index], "day_tot_view": list(per_day_views)})

model = smf.ols('tot_view ~ C(weekday)', data = weekday_views).fit()
day_views['adjusted_day_tot_view'] = day_views['day_tot_view'].mean() + model.resid

# plot the views per day
print(ggplot(day_views, aes('date', 'day_tot_view')) + \
    geom_line(aes(group = 1)) + \
    scale_x_datetime(breaks=date_breaks('3 months'), labels=(date_format('%Y-%m'))) + \
    labs(x='Date', y='Views') + \
    theme(figure_size=(8, 6)) + \
    ggtitle('Original'))

# plot the de-seasonified views per day
print(ggplot(day_views, aes('date', 'adjusted_day_tot_view')) + \
    geom_line(aes(group = 1)) + \
    scale_x_datetime(breaks=date_breaks('3 months'), labels=(date_format('%Y-%m'))) + \
    labs(x='Date', y='Views') + \
    theme(figure_size=(8, 6)) + \
    ggtitle('Day-of-the-week Effects Removed'))
